# gta2man
A collection of tools to manage your gta2 installation.

# Build
`$ make build`

# Test
`$ make test`

# On GTA2
## registry
`HKEY_CURRENT_USER\Software\DMA Design Ltd\GTA2`

## filexts
```
*.gmp - maps
*.gxt - texts
*.mmp - multiplayer map definition
*.mis - plaintext scripts
*.scr - compiled scripts
*.sty - graphics
```
